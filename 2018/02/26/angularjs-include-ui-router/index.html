angularjs页面分割和路由（ui-router)

发表于 2018-02-26 | 分类于 前端 ， JS |

做页面，经常会遇到这样的需求：

  1. 页面太大，需要分割；
  2. 单页面众多状态（页面）切换；

在angular框架前提下，第1点一般通过ng-include来实现（不存在传递参数、不同状态不同显示等问题），所以多用于页眉／页脚／静态页。第2点一般用ui-router来控制。
